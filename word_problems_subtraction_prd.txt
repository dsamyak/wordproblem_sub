1. EXECUTIVE SUMMARY

This document defines product requirements for the "Word Problems Using Subtraction"
interactive lesson module, delivered as Lesson 3.4 within Intellia SG's Grade 1 Math
program. The module targets Singapore Primary 1 students aged 6–7 and teaches students
to understand, model, and solve subtraction word problems within 20, fully aligned with
the Singapore MOE Mathematics Framework and the Primary 1 Targeting Mathematics
(1A/1B) syllabus.

The product is a standalone web page hosted at:
https://intelliasg.com/courses/grade-1-math/lessons/simple-word-problems/

It is built using React (Vite + JSX, JavaScript/CSS) and designed to strictly mirror
the visual and UX structure established at https://numberbound.vercel.app/ and the
repository https://github.com/dsamyak/numberbound.

Audio narration is delivered exclusively via ElevenLabs (Voice: Alice, Voice ID:
Xb7hH8MSUJpSbSDYk0k2, Model: eleven_multilingual_v2) using a pre-generation + dynamic
fallback pipeline identical to the existing numberbound module. No browser Web Speech
API fallback is used. See Section 13 for full Audio Pipeline specification.

The module follows Intellia's 5-phase learner journey:

  Phase 1 — INTRO    → Lesson welcome & phase map overview
  Phase 2 — WONDER   → Curiosity hook, mystery problem
  Phase 3 — STORY    → Narrative-based concept introduction with Singapore characters
  Phase 4 — SIMULATE → Sandbox-style interactive simulation (3 stations)
  Phase 5 — PLAY     → IntelliPlay™ gamified practice (100 randomised questions)
  Phase 6 — REFLECT  → Journal / LearnFlow AI reflection prompt

2. PRODUCT VISION & GOALS

Vision

To make subtraction word problems intuitive, visual, and memorable for 6–7 year old
Singapore learners by teaching them to read a real-world story, identify the "whole"
and "parts", draw a number bond or bar model, and write the subtraction sentence —
all delivered through animated simulations, Singapore-context narratives, and adaptive
gamified challenge within a consistent Intellia experience.

Goals

Goal                              | Metric
----------------------------------|----------------------------------------------
Learning Completion               | ≥85% of students complete all phases
Practice Engagement               | ≥90% attempt at least 10 practice questions
Score Achievement                 | Average challenge score ≥75% first attempt
Session Duration                  | Average engagement ≥16 minutes per session
Curriculum Alignment              | 100% aligned to Singapore MOE Primary 1 (3.4)
Phase Progression                 | ≥80% reach Play phase in a single session
Audio Narration Completion Rate   | ≥90% of students hear all phase narrations

6. PHASE 3 — SIMULATION DESIGN (Detail)

6.1 Station A — "Read & Show" (Concrete)

Context: A word problem is displayed with full narration (ElevenLabs)
Visual:  Large colourful emoji counters in a grid (max 20)
         Singapore-themed emojis: 🍱 🍢 🎈 🏷️ 📚 🐟 🍎 🌟

Interaction:
  Step 1 — Mascot reads problem aloud (ElevenLabs narration)
  Step 2 — "Show all [X] items" — student places counters on board
  Step 3 — "Take away [Y]" — student taps to remove counters
           (removed counters animate a "fly away" disappear effect)
  Step 4 — "How many are left?" — student counts remaining
  Step 5 — Number bond diagram fills automatically showing result

Teaching: Child experiences subtraction kinaesthetically before abstracting

Validation:
  - Must correctly remove the right number before advancing
  - Incorrect: Gentle shake + "Try again! How many should fly away?"
  - After 3 fails: Mascot helps by auto-animating the removal

6.2 Station B — "Draw the Bond" (Pictorial)

Context: Word problem shown + partially completed number bond
Visual:  Interactive number bond SVG with draggable number tiles

Interaction:
  - Three number tiles shown at bottom: whole, part1, part2 (shuffled)
  - Student drags each number into the correct circle in the bond:
      Top circle = Whole (minuend)
      Bottom-left = Given part (subtrahend)
      Bottom-right = Missing part (difference)
  - Each correct placement: bounce animation + tick sound
  - Bar model visual updates automatically alongside the bond

Variants:
  - Variant A: All three numbers shuffled, student places all
  - Variant B: Whole and one part given, student places the third
  - Variant C: Two parts given, student finds and places the whole

6.3 Station C — "Write the Sentence" (Abstract)

Context: Completed number bond shown + word problem above
Visual:  Three blank boxes: [ ___ ] – [ ___ ] = [ ___ ]

Interaction:
  - Tap each blank → NumberPad opens
  - Student fills whole → part1 → part2
  - On completion: animated number sentence appears in full
  - "Answer sentence" field: student selects the correct completion
    e.g. "Wei Ming has [___] fishballs left." → tap correct number

Hint Button: "Show me the number bond" → bond diagram highlights

7.5 Singapore MOE Word Problem Format

Standard format (as per Targeting Mathematics 1A/1B):
  "[Character] had/has [whole] [objects].
   [He/She] [action verb — gave/ate/lost/used/broke] [part1] [objects].
   How many [objects] did [he/she] have left?"

Extended format (harder questions, within 20):
  "[Character A] and [Character B] had [whole] [objects] altogether.
   [Character A] had [part1]. How many did [Character B] have?"

8. GAMIFICATION DESIGN

8.1 Reward System

Stars (⭐):    Earned per 10-question world (1–3 stars based on score)
              ⭐⭐⭐ = 9–10 correct | ⭐⭐ = 7–8 | ⭐ = 5–6 | 0 = < 5
XP Points:    10 XP correct first try
               7 XP correct second try
               5 XP correct with hint used
Streak 🔥:    Fire counter for consecutive correct answers
Streak Bonus: +5 XP per correct answer when streak ≥ 5

8.2 Badges (Unlockable)

🏅 "Problem Spotter"    — Complete Wonder + Story phases
🥈 "Model Builder"      — Complete Simulate phase (all 3 stations)
🥇 "Word Problem Whiz"  — Score 80%+ on Play phase
💎 "Perfect Story"      — Score 10/10 in any world
🔥 "Streak Legend"      — Achieve a streak of 10+ consecutive correct
🌟 "Full Champion"      — Complete all 5 phases (lesson complete)
🦁 "Singapore Solver"   — Complete all 10 Singapore-context worlds
📚 "Bond & Story Link"  — Use the number bond hint 0 times in a world

8.3 Feedback Mechanics

✅ Correct:   Bounce animation + celebration sound + ElevenLabs praise:
               "Amazing! You solved the word problem! 🎉"
               "Brilliant! You used the number bond perfectly! ⭐"
               "You're a maths superstar! Keep going! 🌟"

❌ Incorrect:  Gentle shake + ElevenLabs encouragement:
               "Let's try again! Read the problem carefully. 🔢"
               "Almost there! Think about what we need to find. 🤔"

Hint Level 1: Number bond diagram highlighted with labels
              ElevenLabs: "Look at the number bond. What is the whole?"

Hint Level 2: Counter animation shows take-away step by step
              ElevenLabs: "Count the [objects]. Take away [Y]. Count what's left."

After 2 fails: Answer revealed with animated explanation panel
               ElevenLabs narrates full solution step by step

No negative scoring. Encouragement-first approach always applied.

8.4 Mascot (LearnFlow AI Companion)

Character:  LearnFlow — Friendly Intellia robot (matching brand identity)
Mood States: idle | curious | happy | thinking | celebrating | helping
Appears in: Wonder hook, all Story panels, Simulate stations, Reflect phase
Reacts to:  Correct answer, wrong answer, badge unlock, streak milestone
Narrated:   All mascot speech delivered via ElevenLabs Alice voice

8.5 Progress Indicators

Top bar:      5-dot phase tracker (coloured dots, current phase highlighted)
Bottom bar:   XP counter | Stars total | Streak fire | Phase nav arrows
World map:    Horizontal scrollable map; locked worlds shown as silhouettes
